搭建博客（hexo)

通过昨天的努力，今天算是真正完成了博客的搭建，很方便的提高自己的写作能力。当然，如果按照现在的进程，能够说得是，我现在什么都还没有开始工作。这是一个问题将Hexo的博客搭建流程写来。做一个备忘录。

安装Hexo之前，需要先安装Node.js, Git 和注册GitHub账号。
其次，使用基本的命令： hexo init; Hexo install, Nmp install hexo-deployer-git –save.

每次上传文档：
Hexo g -d.

之后，还可以进行Hexo的主题修改。目前还没有测试过。就这样吧。
